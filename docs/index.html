🔐 SF-Encryptor

A secure file encryption and management tool built with Python and PyQt6. Military-grade encryption with a modern interface.

  • AES-256-CBC, ChaCha20-Poly1305, and Fernet encryption
  • Plugin-based architecture
  • File integrity verification (MD5/SHA-1/256/512)
  • Key generation and management
